    ## Діаграма послідовності (Sequence Diagram)
Ця діаграма показує взаємодію між оператором, базою даних та ML-алгоритмом, який ти досліджуєш у дипломі.

```mermaid
sequenceDiagram
    participant Op as Оператор
    participant DB as База даних Delta
    participant ML as ML-Модуль (Аналіз)
    participant An as Аналітик

    Op->>DB: Внесення координат цілі
    DB->>ML: Запит на перевірку достовірності
    ML-->>ML: Обробка алгоритмом
    ML->>DB: Оновлення статусу (Ймовірність % )
    DB->>An: Сповіщення про нову ціль
    An->>DB: Підтвердження/Відхилення

    stateDiagram-v2
    [*] --> Отримання_даних
    Отримання_даних --> Автоматична_фільтрація
    Автоматична_фільтрація --> Перевірка_ML: Дані коректні
    
    state Перевірка_ML {
        [*] --> Розрахунок_надійності
        Розрахунок_надійності --> Порівняння_з_порогом
    }
    
    Перевірка_ML --> Візуалізація_на_мапі: Надійність висока
    Перевірка_ML --> Додаткова_перевірка: Надійність низька
    
    Візуалізація_на_мапі --> [*]
